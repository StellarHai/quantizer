# A股量能策略系统使用指南

> 基于当前最新模型（ATR动态止损版）编写，涵盖架构、参数、回测、实盘全流程。

---

## 一、系统概览

### 两大使用场景

| 场景 | 脚本 | 说明 |
|------|------|------|
| 历史回测验证 | `run_backtest.py` | 用历史数据验证策略，输出绩效报告 |
| 每日实盘策略 | `daily_strategy.py` | 实时扫描全市场，输出今日买卖建议 |

### 系统架构

```
┌─────────────────────────────────────────────────────┐
│  入口层                                              │
│  run_backtest.py    → 历史回测                       │
│  daily_strategy.py  → 每日实盘策略                   │
└─────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────┐
│  数据层                                              │
│  MarketDataEngine   → 全市场实时行情+批量K线          │
│  RealtimeDataFetcher→ 实盘实时数据（多源降级）        │
│  QuantDataCache     → SQLite本地缓存                 │
│  tools/output/data_cache/ → 本地parquet历史缓存      │
└─────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────┐
│  量化分析层                                          │
│  VolumeQuantizer      → 量能指标计算                 │
│  VolumeSignalGenerator→ 买卖信号生成（-100~+100）    │
│  VolumeRiskManager    → 风险评分与头寸管理            │
│  VolumeStockSelector  → 综合评分选股                 │
└─────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────┐
│  止损层（ATR动态止损）                               │
│  ATRStopLoss          → ATR动态止损/目标价计算        │
│  ImprovedStopLossStrategy → 追踪止损+时间止损        │
└─────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────┐
│  输出层                                              │
│  BacktestAnalyzer     → 回测结果分析导出              │
│  daily_reports/       → 每日策略报告                  │
│  back_test_result/    → 回测图表与数据               │
└─────────────────────────────────────────────────────┘
```

## 二、数据准备（首次使用必读）

### 构建本地历史缓存

回测依赖本地 parquet 缓存，首次使用或需要更新到最新日期时运行：

```bash
python tools/build_stock_cache.py
```

缓存位置：`tools/output/data_cache/{code}_local.parquet`

> 注意：缓存日期范围必须覆盖回测区间，否则报「有效交易日不足」错误。

### 数据源优先级

| 优先级 | 数据源 | 场景 |
|--------|--------|------|
| 本地parquet | 最优先 | 回测时使用 |
| efinance | P0 | 实盘实时行情 |
| akshare东财 | P1 | 备用 |
| akshare新浪 | P2 | K线备用 |

## 三、历史回测

### 快速开始

```bash
# 回测2024全年，均衡策略
python run_backtest.py --start 2024-01-01 --end 2024-12-31

# 近5年全量主板（取前100只）
python run_backtest.py --start 2020-01-01 --end 2025-12-31 --full-market --top-n 100

# 指定股票池
python run_backtest.py --start 2024-01-01 --stocks 600519,000001,600036

# 2026年至今
python run_backtest.py --start 2026-01-01 --end 2026-03-20 --full-market --top-n 100
```

### 完整参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--start` | 必填 | 回测开始日期 YYYY-MM-DD |
| `--end` | 今天 | 回测结束日期 |
| `--capital` | 1,000,000 | 初始资金 |
| `--strategy` | balanced | aggressive / balanced / conservative |
| `--max-positions` | 5 | 最大同时持仓数 |
| `--max-pos-pct` | 0.20 | 单股最大仓位比例 |
| `--min-signal` | 50 | 最小买入信号强度 |
| `--max-risk` | 50 | 最大允许风险评分 |
| `--time-stop` | 10 | 时间止损天数 |
| `--use-atr-stop` | True | 启用ATR动态止损 |
| `--no-atr-stop` | - | 禁用ATR止损（用固定止损） |
| `--atr-period` | 14 | ATR计算周期 |
| `--atr-multiplier` | 2.0 | ATR倍数 |
| `--use-trailing-stop` | True | 启用追踪止损 |
| `--no-trailing-stop` | - | 禁用追踪止损 |
| `--full-market` | False | 全量主板模式 |
| `--top-n` | 100 | 全量模式取前N只 |

### 输出文件

每次回测自动保存到 `back_test_result/{开始日期}_to_{结束日期}_{股票数}stocks/`：

```
├── 01_净值曲线.png
├── 02_年度收益.png
├── 03_回撤曲线.png
├── 04_交易盈亏.png
├── 回测报告.txt       # 完整文本报告
├── 年度数据分析.txt
├── 指标汇总.txt       # 关键指标一览
├── 交易分析.txt
└── trades.csv         # 全部交易记录
```

## 四、每日实盘策略

### 快速开始

```bash
# 扫描今日全市场，输出买入候选（建议收盘后或次日开盘前运行）
python daily_strategy.py

# 导出报告到 daily_reports/{日期}/ 目录
python daily_strategy.py --export

# 同时发送飞书通知
python daily_strategy.py --notify

# 只看前30只
python daily_strategy.py --top 30
```

### 持仓管理

```bash
# 新增一笔持仓（买入后记录）
python daily_strategy.py --add-pos 600519 贵州茅台 1800.0 100 1750.0 1900.0
#                                  代码     名称    买入价 股数  止损价  目标价

# 查看持仓追踪报告（显示每只持仓的止损/止盈状态）
python daily_strategy.py --positions

# 平仓后删除记录
python daily_strategy.py --remove-pos 600519
```

### 完整参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--top` | 20 | 输出前N只候选股 |
| `--min-signal` | 50 | 最小信号强度 |
| `--max-risk` | 50 | 最大风险评分 |
| `--export` | False | 导出CSV和文本报告 |
| `--notify` | False | 发送飞书通知 |
| `--positions` | False | 查看持仓追踪 |
| `--date` | 今天 | 指定分析日期 |

### 输出文件

```
daily_reports/
└── {YYYYMMDD}/
    ├── daily_strategy_{日期}.txt  # 文本报告
    ├── candidates_{日期}.csv      # 候选股CSV
    └── market_state_{日期}.txt    # 大盘状态
daily_reports/positions.csv        # 持仓记录（持久化）
```

### 运行时间建议

| 时间 | 操作 |
|------|------|
| 收盘后15:30~16:00 | 运行 `daily_strategy.py --export --notify`，获取次日候选 |
| 次日开盘9:30 | 按报告中的买入建议，以开盘价附近挂单 |
| 盘中随时 | 运行 `--positions` 检查持仓止损状态 |
| 每周一次 | 更新本地缓存 `python tools/build_stock_cache.py` |

## 五、量化模型详解

### 5.1 股票过滤规则

| 规则 | 说明 |
|------|------|
| 只保留主板 | 上交所60xxxx、深交所00xxxx |
| 排除科创板 | 688xxx |
| 排除创业板 | 300xxx |
| 排除ST/退市 | 名称含ST或退 |
| 排除涨跌停 | 涨跌幅 ±9.8% 以内 |
| 排除低流动性 | 成交额 < 1000万 |
| 排除仙股 | 价格 < 1元 |
| 量比初筛 | 1.2 ~ 8.0（实盘）/ 1.5 ~ 6.0（回测） |
| 换手率初筛 | 0.5% ~ 20% |

### 5.2 量能指标（VolumeQuantizer）

**RVOL（相对成交量）**
```
RVOL = 当日成交量 / 近20日均量
```

| RVOL | 等级 | 含义 |
|------|------|------|
| < 0.7 | 冷清 | 流动性差，不介入 |
| 0.7~1.5 | 正常 | 等待信号 |
| 1.5~3.0 | 活跃 | 有资金关注 |
| > 3.0 | 极端 | 情绪高潮，谨慎 |

**位置判断（基于近30日高低点）**
```
distance_to_high > 10%          → 低位
distance_to_high 5~10%          → 平台
distance_to_high 0~5% 且 RVOL>1.5 → 突破
其他                             → 高位
```

**量价关系**
```
价格↑ 且 成交量↑ → 价涨量增（最佳信号）
价格↑ 且 成交量↓ → 价涨量缩
价格↓ 且 成交量↑ → 价跌量增（危险信号）
价格↓ 且 成交量↓ → 价跌量缩
```

### 5.3 信号生成（VolumeSignalGenerator）

**进场评分（-100 ~ +100）**

| 维度 | 权重 | 高分条件 |
|------|------|----------|
| 量价关系 | 30% | 价涨量增+80分 |
| 股票位置 | 35% | 突破+70 / 低位+60 |
| RVOL等级 | 20% | 活跃+50分 |
| 均线排列 | 15% | 多头+50分 |

**出场评分（触发卖出）**

| 条件 | 扣分 |
|------|------|
| 高位 且 RVOL>2 | -80 |
| 价跌量增 | -70 |
| 空头排列 | -50 |
| 极端量能 | -40 |

**信号判定**
```
出场评分 < -60  → sell（置信度0.9）
出场评分 < -40  → sell（置信度0.7）
进场评分 > 70   → buy（置信度0.9）
进场评分 > 50   → buy（置信度0.7）
进场评分 0~50   → hold
```

### 5.4 风险管理（VolumeRiskManager）

**风险评分（0-100，越高越危险）**

| 因子 | 最高分 | 触发条件 |
|------|--------|----------|
| 股票位置 | 30 | 高位+30 / 平台+15 / 低位+5 |
| 量能状态 | 25 | 极端+25 / 冷清+15 / 活跃+10 |
| 均线排列 | 20 | 空头+20 / 混乱+10 |
| 量价关系 | 15 | 价跌量增+15 |
| 距离前高 | 10 | <5%加+10 / <10%加+5 |

**头寸管理**

| 风险评分 | 等级 | 建议仓位（最大5%）|
|----------|------|------------------|
| 0-24 | 低 | 5.0% |
| 25-49 | 中 | 3.75% |
| 50-74 | 高 | 2.5% |
| 75-100 | 极高 | 1.25% |

### 5.5 ATR动态止损（核心改进）

```
ATR = SMA(True Range, 14天)
True Range = max(high-low, |high-prev_close|, |low-prev_close|)

止损价 = 买入价 - 1.5 × ATR_multiplier × ATR
目标价 = 买入价 + 3.0 × ATR
追踪止损 = 持仓最高价 - 1.5 × ATR（随股价上涨自动上移）
```

**与固定止损对比（2021-2025回测）**

| 指标 | 固定止损(-3%) | ATR动态止损 |
|------|-------------|-------------|
| 年化收益 | -4.08% | +8.07% |
| 最大回撤 | 23.24% | 3.18% |
| 夏普比率 | -0.87 | 1.49 |
| 盈亏比 | 0.93 | 2.17 |
| 平均亏损 | -4.82% | -1.82% |

### 5.6 综合评分与选股策略

```
综合评分 = 信号强度得分 × 40%
         + 置信度得分   × 20%
         + 风险等级得分 × 40%

信号强度得分 = (signal_strength + 100) / 2   # 0~100
置信度得分   = confidence × 100
风险等级得分 = 低→100 / 中→75 / 高→50 / 极高→25
```

**三种策略对比**

| 策略 | 信号要求 | 风险要求 | 适合人群 |
|------|----------|----------|----------|
| aggressive | buy 且 强度≥min_signal | 无限制 | 激进 |
| balanced | buy 且 强度≥min_signal | 等级低/中 | 大多数 |
| conservative | buy 且 强度≥70 | 等级低 且 置信度≥0.8 | 保守 |
