## 三、第二阶段：市场环境过滤（预期提升最大）

### 3.1 大盘趋势过滤

**核心思想**：只在市场整体向上时做多，空仓规避系统性下跌。

```python
class MarketTrendFilter:
    """
    大盘趋势过滤器
    使用沪深300指数状态决定仓位上限
    """
    def get_market_state(self, hs300_df) -> dict:
        ma20  = hs300_df['close'].rolling(20).mean().iloc[-1]
        ma60  = hs300_df['close'].rolling(60).mean().iloc[-1]
        ma120 = hs300_df['close'].rolling(120).mean().iloc[-1]
        price = hs300_df['close'].iloc[-1]

        if price > ma20 > ma60 > ma120:
            return {"state": "牛市", "position_limit": 1.0}    # 满仓
        elif price > ma20 and price > ma60:
            return {"state": "震荡偏多", "position_limit": 0.7} # 7成仓
        elif price < ma20 and price < ma60:
            return {"state": "熊市", "position_limit": 0.0}    # 空仓
        else:
            return {"state": "震荡", "position_limit": 0.4}    # 4成仓
```

**预期效果**：
- 2022年熊市空仓，避免大幅回撤
- 年化收益率预计提升5-10%
- 最大回撤预计从3.8%降至2%以内

## 四、第三阶段：多因子信号融合

### 4.1 引入筹码分布因子

**当前缺失的关键信息**：筹码集中度、获利盘比例。

```
筹码集中（70%筹码分布在当前价±5%内）→ 上方阻力小，趋势明确
筹码分散（历史价格分布均匀）→ 套牢盘多，压力大

获利盘 < 30% + 量能突破 → 绝佳买点（大量筹码仍套牢，反弹空间大）
获利盘 > 80% + 高位放量 → 危险信号（大量浮盈盘随时出货）
```

### 4.2 资金流向因子

class MoneyFlowFactor:
    """
    大单资金净流入因子
    来源：东方财富Level2资金流向

    def score(self, code, date) -> float:
        # 大单净流入 / 成交额
        big_order_ratio = big_inflow / total_amount
        # 连续3日大单净流入
        consecutive_inflow = all(daily_inflow > 0 for last 3 days)

        if big_order_ratio > 0.1 and consecutive_inflow:
            return +20  # 综合评分加20分
        elif big_order_ratio < -0.1:

### 5.3 动态止盈策略

**当前问题**：目标价固定为「30日高点×1.02」，错过大行情。

```python
# 方案一：分批止盈
def partial_take_profit(position, current_price, entry_price):
    gain = (current_price - entry_price) / entry_price
    if gain > 0.08:   # 涨8%，卖出1/3
        sell_ratio = 0.33
    elif gain > 0.15: # 涨15%，再卖1/3
        sell_ratio = 0.33
    elif gain > 0.25: # 涨25%，清仓
        sell_ratio = 1.0
    return sell_ratio

# 方案二：量能萎缩止盈（最符合量能策略逻辑）
# 连续3日成交量逐步萎缩 且 处于高位 → 主动止盈
def volume_shrink_exit(rvol_history, position_type):
    recent_rvol = rvol_history[-3:]
    shrinking = all(recent_rvol[i] > recent_rvol[i+1] for i in range(2))
    if shrinking and position_type == '高位' and recent_rvol[-1] < 0.8:
        return True  # 触发止盈
    return False
```

## 十、风险提示

```
1. 过拟合风险：历史回测优秀≠未来盈利，参数越多越危险
   → 坚守「样本外验证」原则，至少保留2年数据做验证集

2. 数据偏差：历史数据不含退市股，存在幸存者偏差
   → 引入退市股数据（Tushare有完整退市历史）

3. 容量限制：策略规模越大，对市场冲击越大
   → 当前策略适合资金量 < 500万元
   → 超过后需改用VWAP分批执行

4. 政策风险：A股政策随机性强（印花税、融资融券规则等）
   → 不建议使用杠杆，保持纯做多策略

5. 技术故障：实盘中数据断线、API限流
   → 必须实现完善的异常处理和告警机制
```
